# 🚇 Simulation réseau de transport (C#)

## 📌 Description

Projet réalisé dans le cadre de ma formation en développement.

Il s'agit d’un **exercice récapitulatif en programmation orientée objet (POO)** visant à consolider les bases en C# à travers la simulation simplifiée d’un réseau de transport urbain (Metroville).

---

## 🎯 Objectifs pédagogiques

* Manipuler les concepts fondamentaux de l’OO :

  * encapsulation
  * héritage
  * polymorphisme
  * abstraction
  * interfaces
* Structurer un projet en plusieurs classes
* Travailler avec des collections et des interactions entre objets
* Gérer des exceptions

---

## 🧱 Contenu

Le projet modélise :

* différents types de véhicules (`Bus`, `Tram`, `MetroAutomatique`)
* des arrêts avec des passagers
* une ligne de transport
* une simulation de circulation des véhicules

---

## ⚙️ Fonctionnement

* Les véhicules sont créés et démarrés (polymorphisme)
* Chaque véhicule parcourt les arrêts
* Des passagers montent à chaque arrêt
* L’état des véhicules est affiché
* Gestion d’erreurs (vitesse invalide)
* Utilisation d’une interface (`ILocalisable`)
* Détection d’un type spécifique avec cast sécurisé

---

## 📝 Remarque

Projet volontairement simplifié et contraint par un énoncé académique (logique métier minimale).
